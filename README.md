# AnyRAG

> Bring your own LLM. Bring your own embeddings.

A provider-agnostic Retrieval-Augmented Generation (RAG) framework designed with modular architecture, allowing developers to switch LLMs, embedding models, and vector stores with minimal configuration changes.

> 🚧 This project is currently under active development.

## Planned Features

- Multiple LLM providers (Groq, OpenAI, Ollama, Gemini, Claude)
- Multiple embedding providers
- ChromaDB integration
- PDF ingestion
- Configurable RAG pipeline
- Source citations
- FastAPI backend
- Gradio UI
- Docker support
